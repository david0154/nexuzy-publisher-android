# Nexuzy Publisher Android

Nexuzy Publisher Android is an AI-powered RSS-to-WordPress drafting app.

## Core daily flow
1. Add and store RSS links (multiple feeds supported).
2. Fetch news from all active feeds.
3. Detect **today's top/hot news** and **potential viral news**.
4. Group related topics and follow related stories.
5. Verify with OpenAI, rewrite with Gemini, clean with Sarvam.
6. Save draft locally and push WordPress draft with SEO data.

## What is now supported
- Multi-RSS fetch and de-dup by link.
- Today filter + score-based ranking.
- Top articles list for daily posting.
- Hot-news detection and potential-viral detection.
- Related-news clustering by topic (AI, Finance, Politics, Sports, Entertainment, General).
- WordPress draft push with title/content/tag/category/image + SEO fields.
