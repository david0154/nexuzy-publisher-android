# Privacy Policy - Nexuzy Publisher Android

Last updated: 2026-04-21

## Data handled by app
- API keys entered by user (stored locally in app preferences).
- RSS links and user profile metadata when Firebase sync is enabled.
- Generated news drafts in local device database.

## What is stored in Firebase
- User profile metadata (`uid`, email, display name, profile photo url).
- User RSS links only.
- News body/content is not required to be stored in Firebase by default workflow.

## Third-party services
- Google Firebase Auth / Firestore
- Google Gemini API
- OpenAI API
- Sarvam AI API
- WordPress REST API
